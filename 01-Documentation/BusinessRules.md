PRODUCTION
BR-010
all recorded production must be associated with:
*Date
*Funds
*Lot
*WORK
*Crews
BR-011
 Production must recorded using the unit corresponding to the task
 Example:
 Harvest - crates
 Pruning - plants
 Irrigation - hectares
BR-012
The crates must be sorted according to their destination:
*Exportation
*IQF
*National marked
*Discard
